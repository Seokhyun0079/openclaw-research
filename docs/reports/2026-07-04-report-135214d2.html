박력분 파전 레시피 베이킹파우더

요약（30초）

  • 박력분은 글루텐이 적어 파전이 부드럽지만, 베이킹파우더 소량을 넣으면 겉이 더 바삭해집니다
  • 황금 비율：박력분 1컵 + 베이킹파우더 1/2~1작은술 + 찬물 2/3~3/4컵
  • 더 바삭하게：감자전분 1~2큰술 추가 또는 탄산수로 물 대체
  • 핵심：반죽을 오래 치대지 않기 · 기름 넉넉히 · 센 불로 시작

왜 박력분 + 베이킹파우더인가?

재료역할
박력분글루텐 함량 낮음 → 반죽이 덜 질겨짐. 파전이 눅눅해지는 것을 어느 정도 방지
베이킹파우더가열 시 CO₂ 기포 발생 → 반죽 안에 미세 공기층 → 겉바삭·속부드러움
찬물·탄산수글루텐 형성 억제 + 수분 증발 촉진 → 바삭함 UP
감자전분（선택）전분이 글루텐을 덮어 바삭한 식감 강화（튀김가루 대용 가능）

기본 레시피 — 박력분 파전（2~3인분）

난이도 ★☆☆ · 15분 · 파전 2~3장

재료

재료양비고
박력분1컵（120g）없으면 중력분 7 : 박력분 3 혼합
베이킹파우더1/2~1작은술（2~4g）과하면 쓴맛·공허함 → 반절부터
소금1/2작은술—
찬물（또는 얼음물）2/3~3/4컵（160~180ml）박력분 1컵 기준
대파（또는 쪽파）4~6대5~7cm로 송송 썰기
계란1개（선택）고소함·부드러움 UP
식용유넉넉히팬 바닥 2~3mm
참기름약간（선택）마무리용

반죽 농도

숟가락으로 떠서 흘러내릴 때 줄기처럼 이어지는 정도（요구르트보다 약간 묽게）. 너무 되직하면 안 익고, 너무 묽으면 흐물거립니다.

만드는 법

  1. 대파를 씻어 물기를 완전히 제거하고 송송 썹니다.
  2. 볼에 박력분 + 베이킹파우더 + 소금을 먼저 섞어 가루를 고루 섞습니다.
  3. 계란을 넣었다면 풀어 준 뒤, 찬물을 조금씩 부으며 젓가락으로 저어 반죽을 만듭니다.
  4. 덩어리가 약간 보여도 OK — 10~15초만 가볍게 섞고 오래 치대지 마세요（글루텐 형성 = 눅눅해짐）.
  5. 썬 파를 넣고 2~3번만 섞어 1~2분 재워 둡니다.
  6. 팬을 중불 이상으로 달군 뒤 기름을 넉넉히 둘립니다.
  7. 반죽을 한 국자 떠서 얇게 펴고, 센 불 30초~1분 → 중불로 줄여 1~2분 더 굽습니다.
  8. 뒤집기 직전 가장자리에 기름을 한 번 더 둘러 바삭함을 살립니다.
  9. 반대면도 노릇하게 굽고, 마지막에 참기름 1~2방울.

바삭 MAX 버전 — 박력분 + 베이킹파우더 + 전분

재료양
박력분2/3컵（80g）
감자전분（또는 튀김가루）1/3컵（40g）
베이킹파우더1/2작은술
소금1/2작은술
얼음물（또는 탄산수）3/4~1컵
계란1개
대파4~6대

탄산수를 쓸 때는 무가당·플레인만. 반죽을 너무 오래 저으면 탄산이 빠져 바삭함이 줄어듭니다.

전통 방식 — 파 먼저 깔기

  1. 팬에 기름을 두르고 파를 먼저 깝니다.
  2. 반죽을 얇게 뿌려 파 사이사이로 스며들게 합니다.
  3. 겉이 바삭해질 때까지 굽고 뒤집습니다.

파향이 강하고 식당 스타일에 가깝습니다.

실패 없는 꿀팁

  • 베이킹파우더는 가루에 먼저 섞기 — 물에 바로 넣으면 덩어리짐
  • 1/2작은술부터 — 많으면 알칼리 맛·속이 허함
  • 반죽 치대기 금지 — 10~15초 가볍게만
  • 파 물기 제거 — 수분 많으면 눅눅해짐
  • 얇게 부치기 — 두꺼우면 속이 익기 전 겉만 타거나 눅눅
  • 기름 넉넉히 — 바삭함의 핵심（다이어트용이면 에어프라이어 180°C 5~7분）
  • 뒤집기 전 기름 추가 — 전체에 기름 코팅
  • 식었을 때 — 팬에 다시 굽거나 에어프라이어 3~5분이면 바삭함 복구

응용

종류추가 재료
해물파전오징어·새우·조개살 소량 + 청양고추
김치전잘게 썬 김치 1/2컵 — 반죽은 더 되직하게
애호박파전애호박 채썰기 + 당근 약간

해물·김치는 수분이 많으니 반죽을 조금 더 되직하게, 기름을 더 넉넉히 쓰세요.

베이킹파우더 vs 튀김가루

박력분 + 베이킹파우더튀김가루（이미 베이킹파우더 포함）
장점집에 박력분만 있어도 OK바삭함이 안정적, 비율 맞추기 쉬움
주의베이킹파우더 양 조절 필요없으면 박력분 + 전분 + 베이킹파우더로 대체

반죽 황금 비율（핵심）

부침가루가 없어도 밀가루 + 물 + 소금만으로 충분합니다. 농도가 전부입니다.

재료양（1~2장）비고
밀가루（중력분）1컵（120g）없으면 박력분도 OK
물（찬물·얼음물）2/3컵（160ml）밀가루 대비 약 1 : 1.3
소금1/2작은술（2g）—
계란（선택）1개고소함 UP, 반죽 살짝 묽어짐
농도 체크： 숟가락으로 떠서 흘러내릴 때 줄기처럼 이어지는 정도（요구르트보다 약간 묽게）. 너무 되직하면 안 익고, 너무 묽으면 흐물거립니다.
버전밀가루추가물특징
기본1컵—2/3컵촉촉·고소
바삭2/3컵튀김가루 1/3컵1컵（얼음물）겉바속촉
더 바삭2/3컵감자전분 2큰술2/3컵얇고 바삭

기본 레시피 — 밀가루만

난이도 ★☆☆15분2~3인분

재료

  • 대파（또는 쪽파）4~6대 — 5~7cm 길이로 송송 썰기
  • 밀가루 1컵（120g）
  • 물 2/3컵（160ml, 찬물）
  • 소금 1/2작은술
  • 식용유 넉넉히（팬 바닥 1mm）
  • 참기름 약간（마무리용, 선택）

만드는 법

1
대파를 씻어 물기를 털고 송송 썰어 준비합니다.
2
볼에 밀가루 + 소금을 넣고, 물을 조금씩 나눠 부으며 젓가락으로 저어 반죽을 만듭니다.（거품기 OK）
3
썬 파를 반죽에 넣고 가볍게 섞어 1~2분 재워 둡니다.
4
팬을 중불 이상으로 달군 뒤 기름을 넉넉히 둘릅니다.
5
반죽을 한 국자 떠서 팬에 펴고, 센 불 30초~1분 → 중불로 줄여 1~2분 더 굽습니다.
6
뒤집어 반대면도 노릇하게. 마지막에 참기름 1~2방울.

바삭 버전 — 밀가루 + 튀김가루

겉바속촉 해물·김치 파전에도 응용 가능

재료

  • 밀가루 1컵（100g）
  • 튀김가루 1/2컵（50g）— 없으면 감자전분 2~3큰술
  • 얼음물 1컵（200ml）
  • 계란 1개
  • 소금 1/2작은술
  • 대파 3~4대, （선택）오징어·새우·홍합살
  • 식용유 넉넉히

만드는 법

  1. 해물은 키친타월로 수분을 꼭 짭니다.（기름 튐·눅눅함 방지）
  2. 밀가루 + 튀김가루 + 소금을 섞고, 얼음물 + 계란을 넣어 덩어리 약간 남을 정도 가볍게만 섞습니다.（과하게 저으면 딱딱해짐）
  3. 팬에 기름 넉넉히, 중불~센불로 예열.
  4. 파（+ 해물）를 팬에 깔거나 반죽에 섞어 부칩니다.
  5. 양면 노릇할 때까지 2~3분씩. 뒤집기 전 가장자리에 기름 1큰술 추가 → 테두리 바삭.

전통 방식 — 파를 먼저 깔기

막걸리집 스타일. 파가 팬에 직접 닿아 캐러멜화되면서 향이 더 좋아집니다.

  1. 반죽은 얇게 펴는 용도로만 — 파보다 적게（물 10% 더 넣어 살짝 묽게）
  2. 팬에 기름 → 파를 빈틈없이 깔기
  3. 반죽을 숟가락으로 파 위에 얇게 뿌리기（3~4스푼）
  4. （선택）계란 풀어 위에 뿌리기, 고추 올리기
  5. 한 번만 뒤집어 노릇하게 — 파는 금방 익으니 과익 주의

초간단 한 그릇 버전（15분）

  • 밀가루 220g + 물 400ml + 소금 1/2작은술 + （선택）육수가루 1작은술
  • 파 1그릇 넣고 섞기
  • 팬에 기름, 중불로 양면 3~4분씩

가장 쉽지만 물 비율이 많아 얇게 부치는 것이 포인트입니다.

실패 없는 꿀팁 7가지

문제해결
안 익고 눅눅반죽 너무 되직 → 물 조금 추가. 불 너무 약함 → 처음 센 불
파만 타고 밀가루 덜 익음파를 너무 얇게 X, 반죽을 파에 골고루 묻히기
기름에 들러붙음팬 충분히 예열 + 기름 넉넉히. 반죽에 기름 1큰술 섞어도 OK
딱딱해짐반죽 과하게 저음 → 가볍게만 섞기. 계란 1개 추가
바삭함 부족얼음물 + 튀김가루/전분. 얇게 부치기. 뒤집기 전 기름 추가
밋밋함소금 반죽에 넣기. 초간장 + 참기름 마무리
흐물거림물 줄이거나 밀가루 1~2큰술 추가
3단계 황금 공식： ① 반죽은 요구르트보다 살짝 묽게 → ② 첫 면은 센 불 2분 → ③ 뒤집을 때 기름 1큰술 추가
